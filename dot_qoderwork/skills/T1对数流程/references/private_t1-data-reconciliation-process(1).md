## 六、问题答疑

| 问题 | 答疑 |
|---|---|
| 费用账单税率按照什么规则拆分？ | 费用账单涉及税率税额时，查看费用项目主数据的税率拆分规则。 |
| 所有费用项目如何判断收入、费用、成本？ | 根据费用采集策略判断分摊方式，以及金额用加法还是减法。 |
| 费用账单的费用和 BI 对不上，可能原因是什么？ | 先检查是否使用费用账单的不含税费用，是否需要去税。 |
| BI 费用和弹窗费用为什么取反？ | BI 曾对费用项取反，不含税收入取正，便于业务直接看总数；弹窗直接展示底表数据。后续已修改为和弹窗一致。 |
| 是不是所有费用账单数据都是用负数金额减正数金额后的差额入账？ | 要根据费用编码和拆税规则判断。售后返款是收入拆税规则，平台佣金是费用拆税规则。 |
| 其他税金算哪个指标？ | 其他税金包含水利基金建设和印花税-买卖合同。 |
| 有收入没成本的情况是什么原因？ | 销售订单退款了，成本抵消了，但退款金额不是全退，存在差额退款。 |
| 换货主问题：1 号换货主，为什么 2 号仍显示品牌四部？ | 正向订单按下单时的货主继承，可能 3 月下单、4 月发货仍为原货主；退款订单继承原发货订单货主，3 月品四发货、4 月退款仍取品四。 |

## 七、差异排查清单

1. 先确认 T+1 刷数时间：5 号凌晨 3 点、6 号整月刷数、7 号凌晨 3 点重跑或刷上月最后一天。
2. 确认问题涉及的数据表：V2 净利表、科目分摊成功表、科目分摊失败表、费用账单、财务凭证、费用采集策略表。
3. 检查业务报表账套凭证状态是否已审核或已结账，是否存在下推失败。
4. 检查出纳单据来源为手工录入、批量导入、单据复制的金额是否与集团核算一致。
5. 检查仓储服务费、运杂费、卖家运费、返利费是否重复上传，上传人员是否为往来组人员。
6. 检查凭证详情中手工导入、批量导入、凭证复制的科目是否为采集策略中“是”的科目。
7. 费用账单与 BI 差异优先确认税率拆分、不含税口径、费用归属和正负号。
8. 净利导入类费用确认是否按 6 号 12 点前提供，是否按店铺、月份、费用金额或费比口径上传。
9. 计提类费用确认是否已冲销上月计提，并按实际账单重新按日分摊。
10. 换货主、退款、发货跨月问题要区分下单日期、发货日期和退款继承原订单货主。
