# 修饰词

## 订单功能

| 具体修饰词 | 面向业务释义 | 面向开发释义 |
|---|---|---|
| 一般交易（1） | 除未符合其它订单功能条件外的所有订单 | OMS系统订单：除其它条件外；DMS分销订单：订单类别=普通分销/贴牌代工；来源=无订单发货的发货单默认一般交易 |
| 差价交易（2） | 通过拍差价链接补发产品或配件 | OMS系统订单：网店销售+商品CJ-KBZ/TM-XESK+客服备注含"补发配件/补发产品"，或商品=PDDBSCJ，或商品=CJ-KBZ且客服备注为空 |
| 购物金（3） | 天猫/淘宝购买的店铺充值购物金订单 | 商品编码=GWJSP，且收件地址=不需要收货地址 |
| GBSA（4） | 纯优化订单 | 客服备注含"GBSA"关键字 |
| SA（5） | 拍A发B优化订单 | 客服备注含"SA【"且不含"GBSA"，限制店铺类型≠外部客户网店 |
| 补发货品订单（6） | 错发漏发&付费补发场景 | OMS:订单类别=订单补发/作废补发且货主≠配件且平台=线下/班牛；DMS:订单类别=售后补单/退换补采 |
| 补发配件订单（7） | 售后服务补发配件 | OMS:订单类别=补发订单/作废补发且货主=配件且平台=线下/班牛 |
| 补邮费/差价订单（8） | 消费者补齐邮费/差价 | 商品CJ-KBZ/TM-XESK+客服含"邮费"且不含"补发配件/补发产品"，或含"补货款" |
| 样品订单（9） | 邮寄产品样品 | 订单类别=免费样品出库 |
| 快递取号订单（10） | 拆多包裹预约多个物流单号 | 订单类别=快递使用单 |
| 达人样品（11） | 面向达人邮寄样品 | 订单类别=达人样品出库 |

## 销售模式

| 优先级 | 销售模式 | 面向业务释义 | 面向开发释义 |
|---|---|---|---|
| 2.1 | 非销售业务（1） | 补发/样品/快递取号等功能性订单 | OMS:订单类别=补发货品/补发配件/样品/快递取号/达人样品出库；DMS:订单类别=补发货品订单 |
| 2.2 | 线上直销（2） | 依托电商平台开设店铺，终端消费者直接下单 | OMS:网店销售+自营网店+平台=淘宝/京东/拼多多/快手/抖音/天猫超市/美团/小红书/微信视频号/唯品会/淘工厂/京东分销/WPHZY等；或线下零售+平台≠线下/微信社群 |
| 2.2 | 线上分销（代销）（3） | 授权经销商代销销售 | OMS:外部客户网店+平台≠线下/班牛；或供销店铺+代销类型；多种分销销售单条件 |
| 2.2 | 线上分销（经销）（4） | 线上经销商约定供货价格 | OMS:供销店铺+经销类型+平台≠线下/班牛；或分销销售单+奇门分销店铺+产销三部 |
| 2.2 | 线上代销（5） | 自营网店售卖非佳帮手产品（2022财年起不再开展） | OMS:分销类型=代销+平台=淘宝/京东/拼多多/快手/抖音 |
| 2.2 | 平台入仓分销（8） | 直接向电商平台发货入仓 | DMS:订单功能≠补发货品且店铺平台≠线下 |
| 2.2 | 线下零售/直销（6） | 微信群/门店直接面向消费者 | OMS:线下零售+平台=线下/微信社群 |
| 2.3 | 线下分销（7） | 线下经销商 | DMS:订单功能≠补发货品且平台=线下；OMS:分销销售单+供销店铺+平台=线下 |
| 2.4 | 贴牌代工（9） | 工厂生产后贴其他商家品牌标 | DMS:订单类别=贴牌代工且平台=线上 |

## 销售业务动作

| 修饰词 | 面向业务释义 | 面向开发释义 |
|---|---|---|
| 创建 | 单据在数据库中创建 | 使用创建时间取数 |
| 下单 | 消费者加入购物车准备购买 | 使用下单时间取数 |
| 支付 | 消费者为商品付款 | 使用支付时间取数 |
| 审核 | 客服&系统选择仓库/物流/打包箱并处理异常 | 使用审核时间取数 |
| 发货 | 仓库拣货验货打包称重后快递揽收 | 使用发货时间取数 |
| 完成 | 快递送达顾客签收 | 使用完成时间取数 |

## 是否预售

| 修饰词 | 面向业务释义 | 面向开发释义 |
|---|---|---|
| 预售（1） | 先收集需求再采购销售 | 预售订单=是 |
| 非预售（2） | 现货销售 | 预售订单=否 |

## 出仓方式

| 修饰词 | 面向业务释义 | 面向开发释义 |
|---|---|---|
| 自营仓库发货（1） | 自营仓库发货 | 仓库类别=奇门/普通仓库，仓库功能=销售仓库/暂存仓 |
| 代发仓库发货（2） | 供应商代发仓库发货 | 仓库类别=奇门/普通仓库，仓库功能=代发仓库，仓库≠差价链接仓 |
| 外部仓库发货（3） | 京东委外仓/社团委外仓等 | 仓库功能=委外仓 |
| 无需发货（4） | 补差价/GBSA等功能性订单 | 仓库=差价链接仓，或订单功能=GBSA，或仓库为空 |

## 发货状态

| 修饰词 | 面向业务释义 | 面向开发释义 |
|---|---|---|
| 已发货（1） | 完成仓库作业已出库 | OMS系统订单:订单状态=已发货/已完成；原始子订单:物流名称≠空（美团除外）；DMS:发货单状态=已发货/已收货 |
| 未发货（0） | 未完成仓库作业 | OMS系统订单:订单状态≠已发货/已完成；DMS:发货单状态≠已发货/已收货 |

## 系统订单异常类型

| 修饰词 | 说明 |
|---|---|
| 正常订单 | 无异常 |
| 优化订单 | 备注含GBSA/SA |
| 备注订单 | 有买家留言/客服备注且不含GBSA/SA |
| 外部订单 | 外部商家/委外仓发货 |
| 地址异常订单 | 港澳台/海外地址 |
| 0元异常订单 | 付款总金额为0 |
| 白名单异常订单 | 无符合条件包材 |
| 没有匹配的仓库 | 无法匹配发货仓库 |
| 没有匹配的物流 | 无法匹配物流公司 |
| 代发仓复合订单 | 均为代发商品且种类>1 |
| 风控订单 | 高频恶意退款退货黑名单用户 |
| 虚拟订单 | 均为虚拟商品 |
| 物流限区订单 | 收货地址物流无法配送 |

## 订单来源

| 修饰词 | 说明 |
|---|---|
| API抓取 | OMS系统自动API抓取 |
| 手工建单 | 用户手工创建 |
| 手动抓单 | 用户手动抓取 |
| 班牛创建 | 班牛工单系统同步 |

## 支付状态

| 修饰词 | 面向业务释义 | 面向开发释义 |
|---|---|---|
| 已支付（1） | 全部货品已付款 | 原始订单:支付状态=已支付；DMS:支付方式=现金支付且财审时间≠空 |
| 未支付（0） | 全部货品未付款 | 原始订单:支付状态=未支付；DMS:支付方式=现金支付且财审时间=空 |
| 部分支付（2） | 预售仅支付定金 | 支付状态=部分支付 |
