# 平台链接

## 通用规则

- 所有业务均按已付金额计算支付与发货收入；**只统计销售货品**。
- 实时页面限制展示最近 365 日收入>0 的平台链接数据；历史页面限制展示周期内收入>0 的平台链接数据。
- 平台规格编码取值：
  - `doris_dws_trade_order_goods_index` 字段 `reserved6`
  - `doris_dws_refund_order_index` 字段 `plat_goods_code`

## 今日支付与退款

库表（旧表已停止更新，改用新表）：
- 旧：`doris_dws_trade_order_goods_index_rt`、`doris_dws_refund_order_index_rt`
- 新：`doris_app_real_time_sales_report_rt`

| 指标 | 计算 |
|---|---|
| 今日支付金额（不减退款） | 限制 `country_code=CN`、`plat_code<>线下`、货品类别=销售货品；`sum(sales_amount)` |
| 今日退款金额 | `sum(refund_amount)` |
| 今日支付金额（减退款） | `sum(sales_amount) − sum(refund_amount)` |

通用限定：
1. 排除订单平台：线下、班牛
2. 限制销售货品
3. 订单功能=1 一般交易订单 / 2 差价交易订单 / 8 补邮费、补差价订单
4. 销售模式 ≠ 1 非销售订单；≠ 5 线上代销
5. 退款额外：平台状态=退款成功(6) / 退款处理中(7) `source_refund_plat_status`

## 昨日 / 近30天支付指标

库表：`doris_dws_trade_order_goods_index`，支付时间。

| 指标 | 计算 |
|---|---|
| 昨日支付金额（不减退款）/ 近30天支付金额（不减退款） | `sum(payment)` |
| 昨日支付单量 / 近30天支付单量 | 支付时间下，原始单号去重计数 |

限定同上 1~4。

## 未发货退款

库表：`doris_dws_refund_order_index`，商家退款时间。

- **昨日 / 近30天 未发货退款金额**：`sum(refund_fee)`，限定发货状态=未发货 ＋ 平台状态=退款成功/处理中 ＋ 通用限定 1~4。
- **昨日支付金额（减退款）** = 昨日金额（不减退款，支付时间）− 昨日未发货退款金额。
- **昨日支付金额占比** = 昨日单个平台链接支付金额(已减退款) / 所有平台链接昨日支付金额(已减退款)。

## 已发货收入与退款

库表：`doris_dws_trade_order_goods_index`（发货时间），`doris_dws_refund_order_index`（商家退款时间）。

**昨日 / 近3/7/30/本月/上月/财年/365日 已发货收入（不减退款）、销量（不减退款）、单量、不含税/含税采购成本（不减退款）**：

```
发货收入：sum(payment)
不含税采购成本：sum(不含税采购成本总额)
含税采购成本：sum(含税采购成本总额)
销量：sum(num)
单量：发货时间下，原始单号去重计数
```

限定：排除线下/班牛；销售货品；订单功能=1/2/8；销售模式 ≠ 1、5；发货状态=已发货；当订单功能=一般交易订单时，限制销售出库单 `reserved2` 不为空。

**昨日 / 近3/7/30/本月/上月/财年/365日 已发货退款金额、退款销量、不含税/含税采购成本**：

```
sum(refund_fee)
sum(不含税采购成本总额)
sum(含税采购成本总额)
sum(num)
```

限定：排除线下/班牛；销售货品；发货状态=已发货；订单功能=1/2/8；销售模式 ≠ 1、5；平台状态=退款成功/处理中。

## 收入与销量占比

- **昨日 / 近3/7/30/本月/上月/财年/365日 收入** = 已发货收入（不减退款） − 已发货退款金额。
- **收入占比** = 每个平台链接近 3 天收入 / 全平台链接总收入。
- **昨日 / 近3/7/30/本月/上月/财年/365日 销量(发货量)** = 已发货销量(发货口径) − 已发货退款销量；销量占比同收入占比口径。

## 推广费

库表：`dp_ods.doris_ods_upload_expense_detail`。

- **昨日推广费 / 本月推广费**：仅统计平台链接不为空的推广费用；推广费=expense_detail 表支出 − expense_detail 表收入。
- **昨日推广费比 / 本月推广费比** = 推广费 / 同期收入。
- 限定：`expense_detail` 限制费用项目 cost_code 属于"一级费用类目=推广费用"；费用类型=1 实际发生。

## 毛利率与退款率

- **含税毛利率**：1 − (昨日已发货含税采购成本(不减退款) − 昨日退款含税采购成本) / 昨日收入；近30天版本同口径。
- **含税毛利率（近30天）/ 不含税毛利率 / 去税毛利率（近30天）**：1 − (近30日已发货 含税/不含税采购成本(不减退款) − 近30日退款 含税/不含税采购成本) / 近30日收入；限制销售货品 ＋ 剔除线下、班牛。
- **未发货退款率** = 昨日未发货退款金额 / 昨日支付金额(不减退款)；近30日同口径。
- **已发货退款率** = 昨日已发货退款金额 / 昨日已发货收入(不减退款)；近30日同口径。
- **单均价** = 昨日支付金额(不减退款) / 昨日支付单量，四舍五入保留 2 位；近30日同口径。
