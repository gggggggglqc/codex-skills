## 使用规则台账时的要求

必须按“核算范围 + 单据类型 + 入库/出库原因 + 场景条件”定位规则。不要只引用名词解释，也不要只给价格来源摘要。

每个单据规则至少包含这些字段：是否参与出库成本价计算/参与出库成本计算、是否参与结存成本计算、成本来源名称、取值逻辑或赋值逻辑、参与月末成本价计算的库存数量、参与成本价计算的状态、是否参与收入计算或收入总额计算、是否参与发出总额计算、成本价赋值时机、样品货主或其他特殊说明。

空白字段处理：
- 表格中的空白不等于“否”。空白通常表示原表未给出、沿用上下文、或该字段不适用。
- 如果单据类型为空但原因不为空，按上一条有值的单据类型理解，例如 `其他入库单` 下的多个入库原因、`其他出库` 下的多个出库原因。
- 如果规则明确写“数量金额不参与”“无此类单据”“物料仓无此单据”，则按排除规则处理。
- 如果规则明确写“数量参与，单价为0”或“价格为0”，则数量参与、金额按 0 处理，不能当作排除。

### 工厂名词解释

优先上月出库成本价:
1. 本货主+本仓库上月出库成本价。
2. 本货主+本仓库最新含税含运/(1+供应商税率)。
3. 本货主+其他仓上月出库成本价的平均值。
4. 本货主+其他仓最新含税含运/(1+供应商税率)的平均值。
5. 其他货主+本仓库上月出库成本价平均值。
6. 其他货主+本仓库最新含税含运/(1+供应商税率)。
7. 其他货主+其他仓上月出库成本价的平均值。
8. 其他货主+其他仓最新含税含运/(1+供应商税率)的平均值。
9. 若商品类别=清仓品且商品编码以 `XC` 开头，用商品编码查找入库单类型=其他入库、入库原因=瑕疵品入库、账务时间距离当前时间最近的入库单入库成本价格。
10. 本仓库的手工单价。

优先上月结存单价:
1. 本仓库+本货主的上月结存单价。
2. 本仓库+本货主的上月出库成本价。
3. 本仓库+本货主的手工单价。

非自研品优先最新采购价:
1. 用 sku+仓库+供应商取 SRM 最新含税含运/(1+供应商税率)。
2. 用 sku+仓库取 SRM 最新含税含运/(1+供应商税率)。
3. 用 sku 取 SRM 其他仓的最新含税含运/(1+供应商税率)后的平均值；排除成本价为 0 的记录后再平均。
4. 空。

自研品优先最新采购价:
1. 用货主+供应商+仓库+sku 取 SRM 最新含税含运/(1+供应商税率)，阶梯价格表。
2. 用货主+供应商+sku+其他仓取平均值。
3. 用货主+仓库+sku+其他供应商取平均值。
4. 用其他货主+供应商+仓库+sku 取平均值。
5. 用其他货主+供应商+sku+其他仓取平均值。
6. 用其他货主+仓库+sku+其他供应商取平均值。
7. 用供应商+仓库+sku 取 SRM 最新含税含运/(1+供应商税率)，采购价格表。
8. 用仓库+sku+其他供应商取平均值。
9. 用供应商+sku+其他仓取平均值。
10. 用 sku+其他供应商+其他仓取平均值；排除成本价为 0 的记录后再平均。
11. 空。

### 集团名词解释

优先上月出库成本价:
1. 本货主+本仓库上月出库成本价。
2. 本货主+本仓库最新含税含运/(1+供应商税率)的平均值，阶梯价格表。
3. 本货主+其他仓上月出库成本价的平均值。
4. 本货主+其他仓最新含税含运/(1+供应商税率)的平均值，阶梯价格表。
5. 其他货主+本仓库上月出库成本价平均值。
6. 其他货主+本仓库最新含税含运/(1+供应商税率)，阶梯价格表。
7. 其他货主+其他仓上月出库成本价的平均值。
8. 其他货主+其他仓最新含税含运/(1+供应商税率)的平均值，阶梯价格表。
9. 本仓最新含税含运/(1+供应商税率)的平均值，采购价格表。
10. 其他仓最新含税含运/(1+供应商税率)的平均值，采购价格表。
11. 若商品类别=清仓品且商品编码以 `XC` 开头，用商品编码查找入库单类型=其他入库、入库原因=瑕疵品入库、账务时间距离当前时间最近的入库单入库成本价格。
12. 本仓库的手工单价。

优先上月结存单价:
1. 本仓库+本货主的上月结存单价。
2. 本仓库+本货主的上月出库成本价。
3. 本仓库+本货主的手工单价。

非自研品优先最新采购价:
1. 用 sku+仓库+供应商取 SRM 最新含税含运/(1+供应商税率)。
2. 用 sku+仓库取 SRM 最新含税含运/(1+供应商税率)。
3. 用 sku 取 SRM 其他仓的最新含税含运/(1+供应商税率)后的平均值；排除成本价为 0 的记录后再平均。
4. 空。

自研品优先最新采购价:
1. 用货主+供应商+仓库+sku 取 SRM 最新含税含运/(1+供应商税率)，阶梯价格表。
2. 用货主+供应商+sku+其他仓取平均值。
3. 用货主+仓库+sku+其他供应商取平均值。
4. 用其他货主+供应商+仓库+sku 取平均值。
5. 用其他货主+供应商+sku+其他仓取平均值。
6. 用其他货主+仓库+sku+其他供应商取平均值。
7. 用供应商+仓库+sku 取 SRM 最新含税含运/(1+供应商税率)，采购价格表。
8. 用仓库+sku+其他供应商取平均值。
9. 用供应商+sku+其他仓取平均值。
10. 用 sku+其他供应商+其他仓取平均值；排除成本价为 0 的记录后再平均。
11. 空。

自研品调拨单优先最新采购价:
1. 用调入货主+供应商+调入仓库+sku 取 SRM 中调出月份之前的最新含税含运/(1+供应商税率)，阶梯价格表。
2. 用调出货主+供应商+调出仓库+sku 取调出月份之前最新价格。
3. 用调入货主+供应商+sku+其他仓取调出月份之前最新价格平均值。
4. 用调入货主+仓库+sku+其他供应商取调出月份之前最新价格平均值。
5. 用其他货主+供应商+调入仓库+sku 取调出月份之前最新价格平均值。
6. 用其他货主+供应商+sku+其他仓取调出月份之前最新价格平均值。
7. 用其他货主+仓库+sku+其他供应商取调出月份之前最新价格平均值。
8. 用供应商+调入仓库+sku 取调出月份之前最新价格，采购价格表。
9. 用供应商+调出仓库+sku 取调出月份之前最新价格。
10. 用调入仓库+sku+其他供应商取调出月份之前最新价格平均值。
11. 用供应商+sku+其他仓取调出月份之前最新价格平均值。
12. 用 sku+其他供应商+其他仓取调出月份之前最新价格平均值；排除成本价为 0 的记录后再平均。
13. 空。

非自研品调拨单优先最新采购价:
1. 用 sku+调入仓库+供应商取 SRM 中调出月份之前最新含税含运/(1+供应商税率)，采购价格表。
2. 用 sku+调出仓库+供应商取调出月份之前最新价格。
3. 用 sku+调入仓库取调出月份之前最新价格。
4. 用 sku+其他仓库取调出月份之前最新价格后的平均值；排除成本价为 0 的记录后再平均。
5. 空。

## 价格取不到兜底规则

以下规则用于处理按主取价逻辑取不到成本价的场景。执行时必须按顺序查找，命中即停止；规则写“空”或“0”时，表示允许最终为空或为 0，不能自行取其他价格。

### 新逻辑非自研品优先最新采购价

1. 用 sku+仓库+供应商取 SRM 中最新含税含运/1.13。
2. 用 sku+仓库取 SRM 中最新含税含运/1.13。
3. 用 sku 取 SRM 其他仓的最新含税含运/1.13 后的平均值；排除成本价为 0 的记录后再平均。
4. 若仍取不到，结果为空。

### 新逻辑自研品优先最新采购价

1. 用货主+供应商+仓库+sku 取 SRM 中最新含税含运/1.13，来源为阶梯价格表。
2. 用货主+供应商+sku+其他仓取最新含税含运/1.13 的平均值。
3. 用货主+仓库+sku+其他供应商取最新含税含运/1.13 的平均值。
4. 用其他货主+供应商+仓库+sku 取最新含税含运/1.13 的平均值。
5. 用其他货主+供应商+sku+其他仓取最新含税含运/1.13 的平均值。
6. 用其他货主+仓库+sku+其他供应商取最新含税含运/1.13 的平均值。
7. 用供应商+仓库+sku 取最新含税含运/1.13，来源为采购价格表。
8. 用仓库+sku+其他供应商取最新含税含运/1.13 的平均值。
9. 用供应商+sku+其他仓取最新含税含运/1.13 的平均值。
10. 用 sku+其他供应商+其他仓取最新含税含运/1.13 后的平均值；排除成本价为 0 的记录后再平均。
11. 若仍取不到，结果为空。

### 新逻辑优先上月出库成本价

1. 本仓库的上月出库成本价。
2. 本仓库+本工厂对应供应商最新采购价。
3. 本仓库其他供应商最新采购价平均值。
4. 其他仓的上月出库成本价平均值。
5. 其他仓的最新采购价平均值。
6. 若商品类别=清仓品且货品编码以 `XC` 开头，用商品编码查找入库单类型=其他入库、入库原因=瑕疵品入库、账务时间距离当前时间最近的入库单入库成本价格；如果找不到对应单据，入库成本价格为空。
7. 若商品类别=清仓品但货品编码不满足 `XC` 规则，入库成本价格为空。
8. 本仓库的手工单价。

### 仅工厂单据使用：优先上月结存价

1. 本仓库+本货主的上月结存单价。
2. 本仓库+本货主的上月出库单价。
3. 本仓库+本货主的上月手工单价。
4. 若仍取不到，结果为 0。

### 公司优先上月月末成本价

成本价取值优先级保持不变，兜底顺序如下：

1. 本仓库本公司的上月出库成本价。
2. 本仓库其他公司上月出库成本价的平均值。
3. 其他仓所有公司的上月出库成本价平均值。
4. 本仓库最新采购价。
5. 其他仓的最新采购价平均值。
6. 若货品类别=残次品，用商品编码查找入库单类型=其他入库、入库原因=瑕疵品入库、账务时间距离当前时间最近的入库单入库成本价格；如果找不到对应单据，入库成本价格为空。
7. 若货品类别不等于残次品，入库成本价格为空。
8. 本仓库本公司的手工单价。
