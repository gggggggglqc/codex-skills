# 基础指标 — 单据级

## 原始订单

| 名称 | 释义 | 可搭配修饰词 | 可搭配维度 | 精度 |
|---|---|---|---|---|
| 原始订单数量（暂不开发） | 原始订单的数量 | 销售模式(线上直销/线上分销代销/经销)、销售业务动作(下单/支付)、是否预售、订单功能(一般交易/购物金/GBSA/SA/差价交易/补邮费差价)、支付状态、原始订单类型、原始订单平台状态、原始订单退款状态、发货状态、无效货品属性 | 平台、店铺、货主、分销客户、部门、事业群、公司、省份、城市、区县、达人ID、达人名称、用户ID | 整数 |
| 原始订单货款金额 | 订单下商品下单价*数量的加和 | | | 小数点后5位 |
| 原始订单邮费 | 原始订单的邮费 | | | 小数点后5位 |
| 原始订单商家优惠 | 原始订单的商家优惠 | | | 小数点后5位 |
| 原始订单平台优惠 | 平台优惠（主单粒度，子单无需分摊） | | | 小数点后5位 |
| 原始订单应收金额 | 应向消费者收取的款项（主单粒度） | | | 小数点后5位 |
| 原始订单已付金额 | 支付给商家的金额，含买家支付、平台补贴、支付工具优惠 | | | 小数点后5位 |
| 原始订单买家实付金额 | 买家实际支付的款项，不含平台补贴（主单粒度） | | | 小数点后5位 |
| 原始订单退款金额 | 原始订单的退款金额 | | | 小数点后5位 |

| 系统订单货款金额 | 货款=单价×数量 | | | 小数点后5位 |
| 系统订单邮费 | 系统子单分摊邮费 | | | 小数点后5位 |
| 系统订单商家优惠 | 商家优惠=商家单品优惠+商家分摊优惠 | | | 小数点后5位 |

| 系统订单已付金额 | 支付给商家的金额 | | | 小数点后5位 |
| 税率 | 根据店铺编码绑定公司编码，取公司维表doris_dim_company的税率字段 | | | |
| 系统订单收入（不含税） | 已付金额/(1+税率) | | | |
| 系统订单供货成本 | 所有商品品牌供货成本加和（仅已发货时计算） | | | 小数点后5位 |
| 包材费用 | 包材的品牌供货成本（仅已发货时计算） | | | |
| 系统订单预估物流费用 | 关联销售出库单费用中心计算（仅已发货时计算） | | | 小数点后5位 |
| 系统订单实际物流费用 | 物流商真实费用（主单粒度） | | | |
| 系统订单退款金额 | 系统订单退款金额 | | | 小数点后5位 |
