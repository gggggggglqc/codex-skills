# 货品 / 自研 / 品类 / 外采 / SKU 汇总 五张排行榜

**默认排序**：今日销售金额降序

## 共同备注

- SKU 页面：ToC 业务按已付金额核算；ToB 业务按最新供货价核算。
- 货品 / 品类 页面：同上 + 只统计销售货品数据。
- 是否淘汰品 / 是否瑕疵品 / 是否停止下单品 / 商品类型 / 商品类别 / 是否单品 / 是否删除 / 货品类别 / 是否自研 / 品类类目 / SKU 与货品归属关系 → **统一从动销表取值**。

## 限定逻辑（按页面）

### SKU 页面（库存表 join 动销维表）
- 实时页面：筛选日期 dt-1 对动销表 dt
- 历史页面：所有指标取日期筛选区间结束日期的 dt

下拉商品类型：
- **有效品**：限制逻辑同有效 SKU 数
- **清仓品**：goods_type=1、judge_combine=2、删除状态=0、商品类别=清仓品(item_type)、货品类别=销售货品

### 货品类页面（库存表 join 动销维表）
- 实时：筛选日期 dt-1 对动销表 dt
- 历史：取日期筛选区间结束日期的 dt
- 限制：goods_type=1、judge_combine=2、删除状态=0、货品类别=销售货品

### 品类页面（动销维表）
- 同上时间口径
- 同上限制（不需要库存表）

## 备注

- 品类排行榜：过滤类目是 0 或空的数据。
- 汇总行：除 SKU 排行榜库存相关指标只汇总页面展示的 SKU 外，其他指标汇总行统计全部数据（取平均的指标会展示「-」）。
- 限制 `spu_code` 不为空。
- 页面只展示最近 365 天收入 ≠ 0 的数据。
- 报表历史数据默认展示周期内销量 > 0 的数据。

## 指标明细

### 今日销量
- 库表：`doris_app_real_time_sales_report_rt`（实时表）
- 公式：`sum(paid_num - refund_num)`

### 昨日销量（支付口径）/ 昨日销量占比
- 库表：`doris_app_real_time_sales_report_v1`
- 公式：`sum(paid_num - refund_num)`
- 占比：单个 sku/货品/品类的昨日销量(已减退款) / 所有的昨日销量(已减退款)

### 昨日销量（发货口径）/ 近 3/7/30/本月/上月/财年/365 日销量 / 销量占比
- 库表：`doris_app_report_delivery_v1`
- 公式：`sum(paid_num)`

### 今日金额(减退款) / 昨日金额(减退款) / 占比 / 自研金额
- 库表：`doris_app_real_time_sales_report_rt`
- 不区分：`sum(sales_amount - refund_amount)`
- 区分：
  - `sales_model = 线上分销(代销)`：`sum(estimate_cost - estimate_refund_cost)`
  - 其他模式：`sum(sales_amount - refund_amount)`

### 7 日环比
- 库表：`doris_app_real_time_sales_report_v1`
- 公式：`(过去 7 天支付金额(减退款) − 过去 7~14 天支付金额(减退款)) / 过去 7~14 天支付金额(减退款) × 100%`
- `过去 7~14 天 = 过去 14 天 − 过去 7 天`
- 区分销售模式时，过去 7 天 / 14 天分别按销售模式取值

### 昨日 / 近 3/7/30/本月/上月/财年/365 日 收入 / 自研收入
- 库表：`doris_app_report_delivery_v1`
- 不区分：`sum(sales_amount)`
- 区分：
  - 线上分销(代销)：`sum(estimate_brand_quotation)` 供货总额
  - 其他：`sum(sales_amount)` 销售收入(已减退款)
- 业务释义：
  - 实时列表收入(万) = 发货收入 − 退款收入，回刷近 60 天
  - 净利含税收入(万) = 发货收入 − 退款收入，按天回刷，结账后不再回刷
  - 自研：近 3 日自研发货收入 − 近 3 日自研退款收入

### 昨日 / 近 30 日 含税 / 不含税 采购成本
- 库表：`doris_app_report_delivery_v1`
- 公式：`sum(含税采购成本)` / `sum(不含税采购成本)`

### 近 30 日不含税 / 含税毛利率，上月同期同比
- 区分：`1 − 近 30 日不含税(含税)采购成本 ÷ 近 30 日收入`
- 实时含税：`1 − 近 30 日销售货品含税采购成本(已减退款) / 近 30 日销售货品发货收入(已减退款)`
- 实时不含税：同上换不含税
- 历史口径：把「近 30 日」换成「周期内」

### 收入占比 / 自研占比（昨日 / 近 3/7/30/本月/上月/财年/365 日）
- 自研占比：`近 3 日该品类的自研品发货收入(已减退款) / 近 3 日该品类的发货收入(已减退款)`
- 收入占比：`近 3 日该品类的发货收入 / 近 3 日所有品类的发货收入`

### 近 30 日发货销量
- `sum(paid_num)`

### 近 30 日件均价（元，整数）
- `近 30 日收入 / 近 30 日销量`，四舍五入取整

### 退款相关
- 昨日 / 近 30 日未发货退款 = `sum(undelivery_refund_amount)`
- 昨日 / 近 30 日支付金额(不减退款) = `sum(sales_amount)`（支付 V1）
- 昨日 / 近 30 日已发货退款 = `sum(refund_amount)`
- 昨日 / 近 30 日收入(不减退款) = `sum(sales_amount + refund_amount)`
- 未发货退款占比 = 未发货退款 / 支付金额(不减退款)
- 已发货退款占比 = 已发货退款 / 收入(不减退款)

### 库存金额
- `doris_dws_SKU_stock_index`，限销售仓 + 代发仓
- 销售仓 = `总库存数量 × 不含税月度成本价`（空 / 0 取不含税含运采购价）
- 代发仓 = `总库存数量 × 不含税含运采购价`

### 可发天数
- `总库存数量 / 加权日均出库`，向上取整
- 加权日均出库 = `7 日出库/7×60% + 14 日出库/14×40%`，限销售仓 + 代发仓

### 平均日销（实际日销）
- `加权日均出库`，四舍五入保留整数

### 货品名称 / 品类名（前端不展示）
- 货品：`dim_spu` 维表
- 品类：`doris_app_report_delivery_v1` 二级 / 三级品类名称

### 有效 SKU 数（昨日）
- 库表：库存表 + `doris_dim_sku_dynamic_sales`
- 公式：商品编码去重计数
- 限定：goods_type=1 + judge_combine=2 + 删除状态=0 + 货品类别=销售货品 + 商品类别=正常品

### 有效 SPU 数（昨日）
- 公式：`spu_code` 去重计数；限制 `is_valid_goods_with_sale_goods = 2`
- 业务释义：只要有 1 个 SKU 未被打标，定义为有效货品

### 无效 SPU 数（昨日）
- `is_valid_goods_with_sale_goods = 1`

### 有效 SKU 日产
- 库表：`doris_app_real_time_sales_report_v1` + `doris_dim_sku_dynamic_sales`
- 公式：`有效 sku 近 30 日支付金额(减退款) / 有效 SKU 数(昨日) / 30`，向上取整
- **分子不区分销售模式**

## SKU 排行榜专属指标

### 可售库存
- `sum(authentic_total_stock − authentic_lock_stock)`，限销售仓 + 代发仓

### 在途库存
- `sum(in_transit_stock)`

### 未发库存
- `sum(Undelivered_stock)`

### 可用库存
- `正品总库存(正品可售 + 正品锁定) + 在途库存`

### 可用库存天数
- `可用库存 / 加权日均支付数量`，向上取整
- 加权日均支付数量 = `7 日支付/7×60% + 14 日支付/14×40%`

### SKU 创建日期
- 取 ERP 页面创建日期

### 上架天数
- 库表：`doris_dim_SKU_is_new`
- 公式：以 SKU 汇总排行榜为主表 → 关联维表取首次发货时间 → `(昨日 − 首次发货时间)` 天数

### 目标日销 / 目标月销 / 目标达成率
- 库表：`dp_ods.doris_ods_SKU_estimate_sales`
- 目标日销：`sum(estimate_daily_sales)` 按 SKU 汇总，四舍五入保留整数
- 目标月销：`sum(estimate_daily_sales) × 当月天数`，四舍五入保留整数
- 目标达成率：`本月销量 / 目标月销`，百分比保留两位小数

### 货品级别
- 取数仓货品标签表

### 含税含运成本单价
- 库表：仓库维表 `doris_dim_warehouse` + `dp_dim.doris_dim_srm_billing_purchase_ladder_price`（月度采购阶梯价格表→存货成本价）
- 步骤：
  1. 以 SKU 汇总排行榜的 SKU 信息为主表
  2. 限制仓库条件：`warehouse_use_type in (1,2,4)`，`仓库 not in ('WH0454','WH0455','WH0456','WH0480')`
  3. 用 `筛选月份 + 货主 + goods_code` 关联月度阶梯价格维表，取最低 `tax_freight_price`；多货主 / 多店铺取最低
  4. 4.1 号筛选时若 4 月阶梯价数仓未更新，优先用 3 月数据
  5. 若获取到 0 或失败，用 `筛选日期 + goods_code` 取生效中且最低的 `tax_freight_price`
- 历史：取筛选结束日期对应的加价月份；若空则取生效且最低的含税含运采购价

### 快递费（仅 SKU 汇总排行榜）
- 库表：`doris_dim_SKU`
- 公式：`max(settlement_freight)`；昨日 dt，多条取 max
