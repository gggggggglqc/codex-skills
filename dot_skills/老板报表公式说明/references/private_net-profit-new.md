# 税后净利核算指标（新版 价税分离口径）

> 集团/店铺/渠道/部门列表与弹窗的"净利"取值都取**税后净利**字段，且**限制国内**。新版价税分离的净利需求详见"不含税发货口径需求"链接。

## 顶层公式

```
税后净利 = 利润总额 − 所得税
```

集团/部门/渠道/店铺页面单位：万元（保留 2 位）；其他页面、抽屉趋势图单位：元（保留 2 位）。

## 所得税

非产销：
```
若 利润总额 + 销售费用无票 >= 0：所得税 = (利润总额 + 销售费用无票) × 0.15
若 利润总额 + 销售费用无票 < 0：所得税 = 0
```

产销事业群默认 0（导入数据已提供，不需要额外算）。

部门 / 集团（事业群）/ 渠道下的所得税与财务沟通后采用："按该维度下的店铺利润总额对应的所得税累加"。底部汇总和趋势图都按店铺算好的所得税累加（产销仍为旧逻辑，没有产销店铺所得税）；列表与趋势图保持一致。

## 利润总额（净利润）

> 区分 2 个费用分摊表中"收入/成本/费用"：`doris_dim_fms_support_cost_gather_strategy` 中 `cost_belong=2 成本 / 3 费用` 用减法；`cost_belong=1` 用加法，关联时去重。

```
非产销：V2 收入(不含税) + 2 个费用表的不含税收入
       − V2 不含税成本 − 2 个费用表的成本
       − 2 个费用表的费用 − V2 表的费用
       − 税金及附加 − 其他税金（水利建设基金）− 其他税金（买卖合同）
产销事业群（用导入表）：EP001 不含税收入 − EP003 不含税成本 − EP004 不含税费用
                       （导入费用已包含税金，不再额外算）
```

附加说明：
1. 税金及附加、其他税金单独摘出展示。
2. 科目费用分摊成功表 `no_tax_amount` ＋ 分摊失败表 `no_tax_amount` ＋ V2 净利表的不含税费用（`expense_code` 不在 EP001/002/003/004/005/006/028/029/030/033/034/035 中）。
3. 费用表限制：
   ```
   subject_code NOT IN ('1122.01.01','1504','1101.01','6601.39')
   AND code NOT IN ('CI045','CI168','CI177','CI178','CI170','CI175','CI186','CI188','CI194','CI223',
                    'CI225','CI251','CI263','CI265','CI279','CI601','CI602','CI603','CI654','CI030',
                    'CI256','CI157','CI078','CI258','CI289')
   ```

## 价税分离指标对照（库表：`doris_app_net_profit_check_report_v2`，限制国内）

| 指标 | 取值（除特别说明，均限制国别=国内） |
|---|---|
| 含税收入 | `expense_code='EP001'` 时 `expense_amount` |
| 不含税收入 | `expense_code='EP028'` 时 `expense_amount`；产销取导入表 EP001 |
| 销项税额 | `expense_code='EP029'` 时 `expense_amount`；方法二：含税收入 EP001 − 不含税收入 EP028 |
| 不含税成本 | `expense_code='EP003'` 时 `expense_amount`；产销取导入表 EP003；前端弹窗需标识"不含税成本"，旧值取 EP002 |
| 进项税额（成本） | `expense_code='EP030'` 时 `expense_amount`；方法二：含税成本 EP002 − 不含税成本 EP003 |
| 进项税额（费用） | 科目费用分摊成功表 `tax_amount` ＋ 分摊失败表 `tax_amount` ＋ V2 净利表 EP033 卖家运费税额 ＋ EP034 代发费税额 ＋ EP035 包装耗材费税额 |
| 应交增值税 | 销项税额 − 进项税额（成本）− 进项税额（费用）= V2 EP029 − V2 EP030 − V2 EP033 − V2 EP034 − V2 EP035 − 科目费用表 `tax_amount` − 失败表 `tax_amount` |
| 水利建设基金 | `[EP028 不含税收入 + 主营业务收入(6001.99) + 其他业务收入(6051.03)] × 0.05%`；科目编码 6602.38 管理费用_税金 |
| 印花税买卖合同 | `[EP028 不含税收入 + 主营业务收入(6001.99) + 其他业务收入(6051.03) + 主营业务成本(6401.99) + 不含税成本 EP003] × 0.0003` |
| 销售费用无票 | 凭证导入；财务提供科目编码 6601.98 销售费用_无票 |
| 税金及附加 | 应交增值税 × 0.12 |

费用表限制（适用于"进项税额(费用)"、"应交增值税"、"水利建设基金"、"印花税买卖合同"等指标）：
```
subject_code NOT IN ('1122.01.01','1504','1101.01','6601.39')
AND code NOT IN ('CI045','CI168','CI177','CI178','CI170','CI175','CI186','CI188','CI194','CI223',
                 'CI225','CI251','CI263','CI265','CI279','CI601','CI602','CI603','CI654','CI030',
                 'CI256','CI157','CI078','CI258','CI289')
```

> 主营业务收入(6001.99)、其他业务收入(6051.03)、主营业务成本(6401.99) 来源于"科目费用分摊成功表/失败表"（凭证导入和费用账单），需要在应用层计算。

## 科目费用（一级 ~ 六级）

库表（同旧版）：
1. `doris_dws_finance_cost_sbjct`（分摊成功表）
2. `doris_dws_finance_cost_sbjct_failure`（分摊失败表）
3. `doris_dim_fms_support_subject`（限制昨日）
4. `doris_dim_expense_subject_relation`（限制昨日）
5. `doris_app_net_profit_check_report_v2`（限制国内）

取数三步法与旧版一致。新版**汇总展示各级科目的不含税费用金额**；比率 = 各科目费用金额 / 收入，保留 2 位百分比。

附加说明：
- 限制国内；产销是导入表的 EP004 费用。
- 科目分摊表 / 失败表：share 改为不含税费用 `no_tax_amount`。
- V2 净利表：取 V2 净利表的"不含税费用"（`expense_code` 不在 EP001/EP002/EP003/EP004/EP005/EP006/EP028/EP029/EP030/EP033/EP034/EP035 中）。
- V2 净利表的费用 `expense_code` 集合（旧定义）：`('EP031','EP032','EP036','EP007','EP008','EP009','EP010','EP011','EP014','EP015','EP016','EP017','EP018','EP019','EP020','EP021','EP022','EP023','EP024','EP025','EP026','EP027','EP012')`。**优先使用上面的"排除集合"判断不含税费用**，因为费用项 code 是变动的，业务会增加。
