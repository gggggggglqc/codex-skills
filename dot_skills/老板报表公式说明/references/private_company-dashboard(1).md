# 公司大屏

## 通用规则

- ToC 业务按已付金额核算支付/发货收入；ToB 按最新供货价核算。
- **毛利率只统计销售货品**，其他指标统计全部货品的数据。

## 库存类指标（库表均为 `doris_dws_SKU_stock_index`）

| 指标 | 计算逻辑 | 限定 |
|---|---|---|
| 自营仓库存金额 | 总库存×不含税月末成本价（为空/0 取不含税含运采购价） | 排除货主：样品、耗材、OW018、OW016；仓库类别=奇门仓库；仓库功能=销售仓 |
| 代发仓库存金额 | 总库存×采购成本单价（不含税） | 排除货主同上；仓库功能=代发仓 |
| 全仓库存金额 | 自营仓库存金额 ＋ 代发仓库存金额 | — |
| 清仓品库存金额 | 销售仓清仓品总库存×不含税月度成本价 ＋ 代发仓清仓品总库存×不含税含运采购价；清仓品库存金额 = 淘汰品 ＋ 瑕疵品 ＋ 停止下单品 | 库存表 join 动销维表；goods_type=1、judge_combine=2、删除=0、`item_type`=清仓品、货品类别=销售货品；仓库功能=销售仓、代发仓 |
| 本月清仓品处理金额 | 销售仓 yst_delivery_num×不含税月度成本价 ＋ 代发仓 yst_delivery_num×不含税含运采购价 ＋ 委外仓 yst_delivery_num×不含税月度成本价；页面增加国内外区分 | 库存 join 动销，每天对每天关联；其他限制同上 |

## 有效 SKU / SPU / 日产

- **有效 SKU 数**：商品编码去重计数；条件：goods_type=1、judge_combine=2、删除=0、货品类别=销售货品、`item_type`=正常品。
- **有效 SPU 数**：货品编码 spu_code 去重计数；条件含 `is_valid_goods_with_sale_goods=2`。
- **有效 SKU 日产** = 有效 SKU 近30日支付金额(减退款) / 有效 SKU 数(昨日) / 30，**向上取整**；分子不区分销售模式。

## 发单量与销售金额

- **昨日/本月/财年发单量**：发货时间 = 昨日 / 本月初至昨日 / 财年初至昨日；系统单号去重 ＋ 跨境系统单号去重（2025-07-15 起使用跨境自研系统数据，待开发）。库表：`doris_dws_eb_trade_order_goods_index`、`doris_dws_trade_order_goods_index`、`doris_dws_cbs_trade_order_index`。
- **昨日销售金额** / **今日销售金额**：区分销售模式
  - 线上分销(代销)：`sum(estimate_cost - estimate_refund_cost)`
  - 其他模式：`sum(sales_amount - refund_amount)`
  - 库表：昨日用 `doris_app_real_time_sales_report_v1`；今日用 `..._rt` 实时表。

## 收入指标

库表：`doris_app_report_delivery_v1` ＋ `last_year2022_`（去年表，业态默认 ToB、国别默认国内）；财年=最近 4 月 1 日至昨天，不含当天。

V1 表收入计算：
- 线上分销(代销)：`sum(estimate_brand_quotation)`
- 其他模式：`sum(sales_amount)`
- 临时表：`sum(income)`（已减退款），用于社团 ＋ 京东自营。

| 指标 | 释义 |
|---|---|
| 本月收入 | 本月发货收入 − 本月退款收入 |
| 上月收入 | 上月整月发货收入 − 上月整月退款收入 |
| 财年收入 | 财年发货收入 − 财年退款收入 |
| 去年收入 | 去年发货收入 − 去年退款收入 |
| 同期收入 | 去年财年同期发货收入 − 退款 |
| 去年上月收入 | 去年上月整月发货收入 − 退款 |
| 去年本月收入 | 去年本月整月发货收入 − 退款 |
| 去年同期（万） | 去年财年同期发货收入 − 退款；集团需减掉生产 1/2/3/4 部和包材模具的收入 |

**产销事业群**取导入的收入；**集团**=合计 − 产销事业群导入的生产 1/2/3/4 和包材模具，只算产销 1、2、3。

## 预测与同比

- **本年预测收入** = 本年财年收入 ＋ (去年财年收入 ×(1 ＋ 同期增长率) − 本年财年收入)；临时表 `tmp_date.year_income_rate`。
- **预测增长** = (本年预测收入 − 去年收入) / 去年收入。
- **月同比** = (上月收入 − 同比月收入) / 同比月收入。
- **同期增长率** = (本财年收入 − 去年财年同期收入) / 去年财年同期收入；集团需减工厂一部~四部的收入。

## 自研占比与毛利率

- **月度/年度自研品销售占比** = 本月/本财年自研品销售收入 / 本月/本财年收入；分子分母均区分销售模式；限制"是否自研品=是"；产销事业群分子需把导入的产值也加上。
- **上月 / 月度 / 财年 毛利率（含税）** = 1 − 含税采购成本求和 / 财务销售收入求和；限制销售货品。
- **上月 / 月度 / 财年 毛利率（不含税）** = 1 − 不含税采购成本求和 / 财务销售收入求和；限制销售货品。
