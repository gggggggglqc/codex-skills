# 数仓文档管理规范 v1.0

来源：[钉钉文档](https://alidocs.dingtalk.com/i/nodes/QOG9lyrgJP3PAm3kuvy0z6E3VzN67Mw4?utm_scene=team_space)

## 一、文档管理原则

### 1. 按数据场景建立指标文档

每个指标文档用于说明一个明确的数据场景：销售分析、库存分析、采购分析、财务经营分析、会员分析、商品分析等。一个文档内应完整说明该场景下涉及的指标、口径、字段来源、数据逻辑、使用说明及版本变更情况。

### 2. 一个文档对应一个数据场景

不建议将多个无关场景混在同一个文档中维护。

### 3. 文档分层结构

文档划分为三层：
- **Index 层**：索引入口
- **应用层**：具体指标说明
- **多维数据应用层**：多维分析场景

## 二、单文档管理规范

### 1. 第一个 Sheet 固定为"版本说明"

版本说明 Sheet 至少包含以下字段：

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| 版本号 | 必填 | 与禅道版本号保持一致 |
| 版本内容 | 必填 | 简要说明本期修改内容 |
| 上线状态 | 必填 | 未上线 / 已上线 |
| 上线时间 | 必填 | 实际上线时间 |
| 维护人 | 必填 | 本次文档维护人 |
| 禅道链接 | 必填 | 对应本期需求禅道地址 |
| 备注 | 选填 | 补充特殊说明 |

### 2. 版本号规则

文档版本号必须与禅道版本号保持一致（如 V1.0.0、V1.1.0、V2.0.0）。

### 3. 上线状态

| 状态 | 含义 |
|------|------|
| 未上线 | 文档对应功能尚未上线 |
| 已上线 | 文档对应功能已正式上线 |

功能上线后，维护人需及时更新状态并补充上线时间。

## 三、指标说明 Sheet 管理规范

### 1. 命名规则

```
版本号 + 简述
```

示例：
- `V1.0.0 初始版`
- `V1.1.0 销售额口径调整`
- `V1.2.0 新增商品维度`
- `V2.0.0 经营分析模型重构`

### 2. 最新版本展示规则

文档中默认展示当前最新版本的指标说明 Sheet。每次版本更新后，将历史版本 Sheet **隐藏**（不删除），仅保留最新版本可见。

### 3. 历史版本管理

- 历史版本只允许隐藏，不允许删除
- 已上线的历史版本原则上不得再修改
- 如发现历史版本有明显错误，在当前版本中修正并在版本说明中记录

## 四、本期变更标识规范

### 1. 红色字体标识

每次版本更新时，本期新增、修改或删除的内容在当前版本 Sheet 中使用**红色字体**标识。适用范围：新增指标、指标口径调整、字段来源调整、计算公式调整、过滤条件调整、维度新增或删除、刷新频率调整、使用说明调整、注意事项调整。

### 2. 红色字体使用周期

- 仅标识当前版本的本期变更
- 下一版本发布时，非本期变更内容恢复为正常字体
- 新版本中仅对本期新增或调整内容标红

### 3. 删除内容标识

本期删除的内容不可直接物理删除：
- 在当前版本进行~~画线~~处理
- 最新一期中非本期删除的指标需要删除

## 五、额外说明管理规范

### 1. 额外说明需绑定版本

格式：`【V1.1.0 额外说明】`

### 2. 更新规则

| 情况 | 处理方式 |
|------|----------|
| 仍适用于当前版本 | 更新版本号或保留说明 |
| 只适用于历史版本 | 随历史版本 Sheet 一起隐藏 |
| 已失效 | 标记废弃或删除到历史版本中 |
| 内容发生变化 | 在当前版本中更新，并在版本说明中记录 |

## 六、版本更新操作流程

1. **复制上一版本 Sheet** → 新 Sheet 按"版本号 + 简述"命名
2. **修改本期内容** → 本期变更用红色字体标识
3. **更新版本说明** → 在第一个 Sheet 新增版本记录
4. **检查额外说明** → 确认是否仍适用新版本
5. **隐藏历史版本** → 仅保留最新版本可见
6. **上线后更新状态** → 状态改为"已上线"并补充上线时间

## 七、解析数仓文档时的执行规则

当解析数仓指标文档（Excel/在线表格/钉钉文档）时，必须按以下规则执行：

1. **定位版本说明**：读取第一个 Sheet 的版本说明，确认当前最新版本号
2. **定位最新指标说明**：找到以最新版本号命名的 Sheet（如 `V1.2.0 新增商品维度`）
3. **识别本期变更**：红色字体内容为本期新增或修改项
4. **识别删除项**：~~画线~~内容为本期删除项
5. **读取额外说明**：检查是否有绑定当前版本的额外说明
6. **对比历史版本**：如需追溯口径变化，取消隐藏历史版本 Sheet 对比
7. **口径确认**：每个指标需关注：指标名称、计算公式、字段来源、过滤条件、刷新频率、使用说明、注意事项
